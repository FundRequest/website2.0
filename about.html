WHAT IS FUNDREQUEST?

FundRequest is a monetization platform built on blockchain technology, that allows anybody to fund open issues on any open source project in a fast and easy way. Funding a request takes only a minute, and doesn’t require any coding skills.

With FundRequest, any user can create a contractual reward to solve a problem, fix a bug or implement a feature. Funds are allocated to a specific issue, and when a developer solves the problem or develops the feature, he/she receives the reward.

At first, FundRequest will be integrating with Github and Stack Exchange, but our goal is to evolve into a system that can be used to fund and reward any community platform.

HOW IT WORKS

With FundRequest, users can request and reward any OSS development issue in just a few clicks. Developers can then propose their solution for a feature request or bugfix. Whoever comes up with the best solution gets the reward – it’s safe and simple!
